# 5-text_indentation.txt

===========================
How to Use 5-text_indentation.py
===========================

This module defines a text-formatting function ``text_indentation(text)``.

Usage
=====

The function prints the text with 2 new lines after each of these characters: ., ? and :

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("Hello. How are you? I'm fine: thanks.")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    I'm fine:
    <BLANKLINE>
    thanks.

No spaces are printed at the beginning or end of lines:

::

    >>> text_indentation("   Spaces.   Are removed.   ")
    Spaces.
    <BLANKLINE>
    Are removed.

::

    >>> text_indentation("No punctuation")
    No punctuation

Multiple punctuation marks are handled correctly:

::

    >>> text_indentation("Hello... World? Yes!!!")
    Hello.
    <BLANKLINE>
    .
    <BLANKLINE>
    .
    <BLANKLINE>
    World?
    <BLANKLINE>
    Yes!!!

Invalid Input
=============

The parameter ``text`` must be a string. Otherwise, a TypeError is raised.

::

    >>> text_indentation(42)
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation(["list", "is", "not", "string"])
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation(None)
    Traceback (most recent call last):
    TypeError: text must be a string

At least one argument must be provided:

::

    >>> text_indentation()
    Traceback (most recent call last):
    TypeError: text_indentation() missing 1 required positional argument: 'text'

Empty string input:

::

    >>> text_indentation("")
