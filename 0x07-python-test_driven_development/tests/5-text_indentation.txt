The ``5-text_indentation`` module
============================

Using ``text_indentation``
---------------------

Import the function from the module:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test with no delimiters:
    >>> text_indentation("Hello Holberton")
    Hello Holberton

Test string with trailing spaces:
    >>> text_indentation("Holberton          ")
    Holberton

Test string with leading spaces:
    >>> text_indentation("        Holberton")
    Holberton
    
Test string with both leading and trailing spaces:
    >>> text_indentation("        Holberton          ")
    Holberton
    
Test with an empty string:
    >>> text_indentation("")

Test with a string of spaces:
    >>> text_indentation("      ")

Test with a simple example:
    >>> text_indentation("Hello: Holberton")
    Hello:
    <BLANKLINE>
    Holberton

Test delimiters with spaces:
    >>> text_indentation("      ?:  :  ")
    ?
    <BLANKLINE>
    :
    <BLANKLINE>
    :
    <BLANKLINE>

Test with a newline:
    >>> text_indentation("\n")
    <BLANKLINE>

Test with multiple newlines:
    >>> text_indentation("\n\n")
    <BLANKLINE>
    <BLANKLINE>

Test newline after text:
    >>> text_indentation("Hello\n\n")
    Hello
    <BLANKLINE>

Test multiple delimiters:
    >>> text_indentation("Hello: Holberton? school. fooo")
    Hello:
    <BLANKLINE>
    Holberton?
    <BLANKLINE>
    school.
    <BLANKLINE>
    fooo

Test multiple delimiters with spaces:
    >>> text_indentation("Hello   :     Holberton    ?    school.    fooo")
    Hello:
    <BLANKLINE>
    Holberton?
    <BLANKLINE>
    school.
    <BLANKLINE>
    fooo

Test delimiters with spaces at the end:
    >>> text_indentation("Hello   :     Holberton    ?    school.    fooo     ")
    Hello:
    <BLANKLINE>
    Holberton?
    <BLANKLINE>
    school.
    <BLANKLINE>
    fooo

Test only delimiters:
    >>> text_indentation(".?:")
    .
    <BLANKLINE>
    ?
    <BLANKLINE>
    :
    <BLANKLINE>

Test with just a delimiter and spaces:
    >>> text_indentation("     ?   ")
    ?
    <BLANKLINE>

Test non-indented text after a delimiter:
    >>> text_indentation("?Hello.")
    ?
    <BLANKLINE>
    Hello.
    <BLANKLINE>

Test with existing newlines:
    >>> text_indentation("?\n\n.\n\n:")
    ?
    <BLANKLINE>
    <BLANKLINE>
    <BLANKLINE>
    .
    <BLANKLINE>
    <BLANKLINE>
    <BLANKLINE>
    :
    <BLANKLINE>

    >>> text_indentation(None)

    ...

    >>> text_indentation(33)
